# 🏫 용인대학교 & AI융합학부 가이드 챗봇 (RAG Project)

이 프로젝트는 **용인대학교 학사 규정**과 **AI융합학부 전공 정보**를 학습한 AI 챗봇입니다.
사용자가 질문하면 관련된 학교 문서를 찾아보고, **GPT-4o**가 정확한 답변을 해줍니다.

---

## 🛠️ [필수] 설치 및 실행 가이드
1. 파이썬 3.10.11 버전 설치하기
2. 업로드 한 파일 다운받기 (반드시 하나의 폴더 안에 모두 위치해야 함.)
3. 해당 폴더 안에 가상환경 생성 및 활성화 (.venv)
4. 가상환경 활성화 된 상태에서 "pip install -r requirements.txt"로 필수 라이브러리 설치
5. 공유 된 API Key 설정 후 "streamlit run app.py"로 코드 실행하기
